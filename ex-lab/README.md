## 基于索引分区的内存分配

对应的代码文件为：

- q2.h 头函数，存放函数声明
- q2f.c 源文件，存放函数定义
- q2.c 源文件，存放主函数

使用如下命令可编译可执行文件：

```bash
gcc q2.c q2f.c -o q2
```

最后在命令行中输入`./q2`即可执行编译生成的文件。
